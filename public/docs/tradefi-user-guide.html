TradeFi User Guide Documentation

Designed to help investors trade better

Introduction

TradeFi is a platform aimed at retail investors to demystify the myths. Our objective will be fulfilled through “TradeFi” which is a mobile and desktop web application (app), in which people can become investors by executing trades and investing funds. Currently, there is no trading platform for an individual to trade fixed income securities in the market. TradeFi aims to bridge this gap in the Nigerian financial landscape.

This documentation guide for all TradeFi users, irrespective of their stage on the application i.e Activation or Simulation.


TradeFi Categories

TradeFi has three divisions that enhance the users experience; this will be done by being enlightened about the products that are available, practicing how to trade the products, through a simulator and activating a profile to execute trades.
